# Dragon Squadron Multigame v15

Upload the full contents of this folder to the GitHub repository root and allow Vercel to redeploy.

No Supabase SQL change is required if `dragon_squadron_games` already exists.

v15 changes:
- Player and civilian vessels now draw track numbers from one shared random pool.
- Player ships no longer use distinctive 900-series track numbers.
- Track labels display as plain numbers only, without the `T` prefix.
- Track labels are slightly smaller on the chart.
- Own-ship/vector projections are drawn as a thinner chart-style arrow with cross-ticks at 3, 6 and 9 minutes.
- Radar plot trails are now subdued/dull trails rather than bright dot chains.
