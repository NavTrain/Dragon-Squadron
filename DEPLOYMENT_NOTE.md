# Deployment note

Unzip the package and upload the complete contents to the project root.

Do not upload only index.html. The app needs the PNG files beside it.

Required files include:
- index.html
- peacock_token.png
- plover_token.png
- starling_token.png
- swallow_token.png
- swift_token.png
- event_*.png
- .nojekyll
- supabase.sql
