# Deployment

Unzip this package and upload all files inside the folder to the root of the GitHub repository.

Required files:

- index.html
- peacock_token.png
- plover_token.png
- starling_token.png
- swallow_token.png
- swift_token.png
- token_preview.png
- supabase.sql
- README.md
- DEPLOYMENT_NOTE.md
- .nojekyll

Do not upload the zip file itself. Do not put index.html inside a subfolder.
